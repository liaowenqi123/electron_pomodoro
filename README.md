## 🚀 快速开始

### 环境要求

- **Node.js** >= 16.x
- **npm** >= 8.x

### 安装与运行

```bash
# 1. 克隆仓库
git clone https://github.com/liaowenqi123/electron_pomodoro.git
cd electron_pomodoro

# 2. 安装依赖
npm install

# 3. 添加音乐文件（可选）
# 在 music-player 文件夹下创建 music 文件夹，并放入音频文件
# 支持 .wav, .mp3, .flac, .ogg, .m4a 格式

# 4. 启动应用
npm start
```

### 打包发布

```bash
npm run build
```

打包后生成 `dist/番茄钟-win32-x64/` 目录。

## 📚 功能详解

### 计时器

#### 单次模式

单次模式适合一次专注一个任务。

**操作方式：**
1. **选择预设时间**：点击左侧预设列表中的时间（默认：工作 15/25/45/60 分钟，休息 5/10/15 分钟）
2. **自定义时间**：使用滚轮选择器选择 1-120 分钟，点击 `+` 添加新预设
3. **添加备注**：选择时间后，可在计时器圆圈上方输入备注（标题 + 详细内容）
4. **开始计时**：点击「开始」按钮
5. **暂停/继续**：点击「暂停」暂停，再次点击「继续」恢复（专注模式下禁用暂停）
6. **重置计时**：点击「重置」恢复初始状态

**预设管理：**
- 点击预设右侧的 `×` 删除预设
- 预设数据自动保存，下次启动恢复

#### 工作/休息模式

点击计时器上方的「💼 工作」或「☕ 休息」切换模式：
- **工作模式**：预设时间为 15/25/45/60 分钟，界面显示番茄红色
- **休息模式**：预设时间为 5/10/15 分钟，界面显示清新绿色

### 计划模式

计划模式适合规划一整天的专注任务，系统会自动依次执行。

**操作方式：**
1. **切换到计划模式**：点击顶部「单次 ○──● 计划」滑块
2. **添加任务**：
   - 使用滚轮选择时间
   - 在下方输入备注（标题 + 详细内容）
   - 点击「+ 工作」添加工作时间，或「+ 休息」添加休息时间
3. **调整顺序**：拖拽任务项调整执行顺序
4. **编辑备注**：点击任务项上的 📒 图标查看/编辑备注
5. **删除任务**：点击任务项右侧的 `×`（至少保留一个任务）
6. **开始执行**：点击「开始」按钮，系统自动从第一个任务开始

**计划执行流程：**
```
任务1 → 计时完成 → 任务2 → 计时完成 → 任务3 → ... → 全部完成
```

**注意事项：**
- 计划运行时无法拖拽排序
- 计划运行时无法添加/删除任务
- 点击「重置」会停止整个计划，恢复到第一个任务

### 菜园子系统

菜园子是一个专注激励机制，通过种植作物、收获出售来获得成就感。

#### 基础概念

| 概念 | 说明 |
|------|------|
| **专注模式** | 开启后参与奖惩机制，计时运行时作物成长 |
| **菜园格子** | 12 个格子，初始解锁 6 个，用于种植作物 |
| **种子背包** | 存放购买的种子 |
| **作物背包** | 存放收获的作物 |
| **金币** | 购买种子、出售作物获得 |

#### 作物类型

| 作物 | 图标 | 成长时间 | 稀有度 | 种子价格 | 出售价格 |
|------|:----:|:--------:|:------:|:--------:|:--------:|
| 胡萝卜 | 🥕 | 25 分钟 | 普通 | 8 金币 | 10 金币 |
| 番茄 | 🍅 | 50 分钟 | 普通 | 16 金币 | 20 金币 |
| 向日葵 | 🌻 | 90 分钟 | 稀有 | 40 金币 | 50 金币 |
| 玫瑰 | 🌹 | 120 分钟 | 稀有 | 64 金币 | 80 金币 |
| 金桂树 | 🌳 | 180 分钟 | 传说 | 120 金币 | 150 金币 |

#### 种植流程

```
购买种子 → 选择种子 → 点击空格子种植 → 开启专注模式计时 → 作物成长 → 成熟后点击收获
```

**详细步骤：**
1. 点击顶部「🌱」按钮打开菜园子窗口
2. 点击「🏪 商店」购买种子
3. 在种子背包中点击选择一种种子
4. 点击菜园中的空格子进行种植
5. 回到主界面，开启专注模式并开始计时
6. 计时运行时，作物每分钟增加 1 分钟成长进度
7. 作物成熟后，打开菜园子点击作物收获

#### 商店功能

| 标签页 | 功能 |
|--------|------|
| **购买种子** | 购买各种作物的种子，价格见上表 |
| **出售作物** | 出售已收获的作物，支持单个出售和一键出售全部 |

#### 惩罚机制

专注模式下，以下行为会导致所有正在生长的作物**枯萎死亡**：

| 行为 | 说明 |
|------|------|
| 点击「重置」按钮 | 中断当前专注 |
| 检测到娱乐前台 × 3 次 | 前台检测累积 3 次警告 |
| 关闭应用窗口 | 专注模式下关闭窗口需确认 |

**保护机制：**
- 专注模式下禁用「暂停」按钮
- 专注模式下禁止种植新作物
- 关闭窗口前弹出确认对话框

### 专注模式

专注模式与前台检测联动，防止你在专注时分心刷娱乐应用。

#### 开启专注模式

1. 点击计时器下方的「专注模式」开关
2. 开启后状态显示「开启」，开关变为激活状态
3. 开始计时后，前台检测自动启动

**注意：** 计时运行中无法切换专注模式状态。

#### 前台检测逻辑

```
检测到前台窗口变化
    ↓
匹配白名单？ ──是──→ 通过（不警告）
    ↓否
匹配黑名单？ ──是──→ 警告
    ↓否
查询历史记录？ ──有记录──→ 根据历史判断
    ↓无记录
调用 DeepSeek AI 判断 ──是娱乐──→ 警告
    ↓不是娱乐
通过（记录到历史）
```

#### 警告弹窗

检测到娱乐前台时弹出警告：

```
┌─────────────────────────────────┐
│           ⚠️                    │
│     检测到娱乐前台               │
│   【B站 - 某视频标题】           │
│     警告次数：1/3                │
│                                 │
│ [不是娱乐]      [知道了]         │
└─────────────────────────────────┘
```

**按钮说明：**
- **不是娱乐**：将当前窗口标记为非娱乐，以后不再警告
- **知道了**：承认警告，累积警告次数

#### 惩罚触发

累积 **3 次**警告后触发惩罚：
1. 所有正在生长的作物枯萎
2. 计时器重置
3. 专注模式关闭
4. 弹出系统通知

#### 黑白名单管理

| 配置文件 | 路径 | 内容 |
|----------|------|------|
| `list_config.json` | `foreground_inspection/` | 白名单、黑名单、历史记录 |

**默认白名单：**
- 文件资源管理器
- Visual Studio Code
- PowerShell
- PowerPoint
- 演示文稿
- 番茄钟

**默认黑名单：**
- bilibili
- 抖音
- 快手
- 斗鱼
- 虎牙

### 音乐播放器

内置音乐播放器，支持边工作边听音乐。

#### 基本操作

| 操作 | 方式 |
|------|------|
| **播放/暂停** | 点击 `▶` 按钮 |
| **上一首** | 点击 `⏮` 按钮（需要播放过上一首才可用） |
| **下一首** | 点击 `⏭` 按钮 |
| **跳转进度** | 点击或拖动进度条 |
| **调节音量** | 点击 `🔊` 按钮展开音量滑块 |
| **切换设备** | 点击 `🎧` 按钮选择输出设备 |
| **收起/展开** | 点击播放器左侧的 `▼` 按钮 |

#### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `右 Ctrl + 右 Shift` | 播放/暂停 |
| `右 Ctrl + ←` | 上一首 |
| `右 Ctrl + →` | 下一首 |
| `右 Ctrl + ↑` | 音量增加 |
| `右 Ctrl + ↓` | 音量减少 |

#### 添加音乐

将音乐文件放入 `music-player/music/` 文件夹：
- 支持格式：`.wav`、`.mp3`、`.flac`、`.ogg`、`.m4a`
- 自动扫描并随机播放

#### 输出设备切换

点击 `🎧` 按钮选择音频输出设备：
- 默认设备会标记 `✓`
- 切换设备后设置自动保存
- **注意**：除非你清楚自己在做什么，否则请勿更改

### AI 规划助手

使用自然语言描述你的需求，AI 自动生成番茄计划。

#### 使用条件

- 需要 **Admin 用户** 登录
- Admin 用户自动获取 DeepSeek API Key

#### 使用方法

1. 点击顶部「🤖」按钮打开 AI 助手
2. 在输入框中描述你的需求，例如：
   - "我要学习 3 小时"
   - "准备考试，每学习 45 分钟休息 10 分钟"
   - "写代码 2 小时，中间休息两次"
3. 点击「✨ 生成计划」
4. 查看生成的计划，点击「✅ 应用计划」添加到计划模式

#### 生成示例

**输入：** "明天要考试，需要复习数学 3 小时"

**AI 输出：**
```
📋 为您规划了 3 小时的数学复习计划

1. 💼 工作 45分钟 - 复习数学第一部分
2. ☕ 休息 10分钟
3. 💼 工作 45分钟 - 复习数学第二部分
4. ☕ 休息 10分钟
5. 💼 工作 45分钟 - 复习数学第三部分
6. ☕ 休息 15分钟

⏱️ 总计: 3小时5分钟
```

### 数据统计

记录并可视化你的专注数据。

#### 查看统计

点击顶部「📈」按钮打开统计弹窗。

#### 时间范围

| 范围 | 说明 |
|------|------|
| **每日** | 最近 7 天的数据，按日显示 |
| **每周** | 最近 4 周的数据，按周显示 |
| **每月** | 最近 6 个月的数据，按月显示 |

#### 图表类型

| 类型 | 说明 |
|------|------|
| **柱状图** | 堆叠显示不同备注类别的时长 |
| **折线图** | 显示专注时长趋势 |
| **饼图** | 显示各备注类别的时长占比 |

#### 统计内容

- **专注次数**：完成的番茄钟数量
- **专注时长**：累计专注分钟数
- **平均时长**：每次专注的平均时长
- **详细数据表格**：每日/周/月的具体数据

### 云端登录

支持用户注册登录，Admin 用户可使用高级功能。

#### 登录注册

点击顶部「☁️」按钮打开登录窗口：

| 操作 | 说明 |
|------|------|
| **注册** | 输入用户名（至少 2 字符）和密码（至少 6 位） |
| **登录** | 输入用户名和密码 |
| **记住密码** | 下次打开自动填充账号密码 |
| **自动登录** | 下次打开自动登录 |

#### Admin 权限

Admin 用户享有以下特权：
- 自动获取 DeepSeek API Key（仅内存存储，不持久化）
- 可使用 AI 规划助手
- 前台检测使用 AI 判断，准确率更高

**获取 Admin 权限：**
1. 在 Supabase Dashboard 中打开 `users` 表
2. 找到你的用户记录，将 `admin` 字段设置为 `true`
3. 重新登录即可

#### 安全说明

- API Key 仅存储在内存中，不进行本地持久化
- 退出登录后 API Key 自动清除
- 密码使用 PBKDF2 + SHA512 哈希存储

支持明暗主题切换，保护眼睛。

点击顶部展开按钮，然后点击「🌙」/「☀️」按钮切换：

### 迷你模式

计时运行时最小化窗口进入迷你模式。

#### 进入迷你模式

计时运行时点击「−」最小化按钮：
- 窗口缩小为桌面小组件
- 始终置顶显示
- 显示圆形进度条和时间
- 可拖动到任意位置

#### 退出迷你模式

- 点击迷你窗口底部的「⬆」按钮
- 或通过系统托盘右键菜单选择「展开窗口」

#### 系统托盘

迷你模式下会在系统托盘显示图标：
- **左键点击**：显示/聚焦窗口
- **右键菜单**：
  - 展开窗口
  - 退出应用

#### 位置记忆

迷你模式的位置会被记住，下次进入时恢复到相同位置。

### 备注功能

为每个番茄钟添加备注，方便记录工作内容。

#### 单次模式备注

1. 选择时间后，在计时器圆圈上方输入备注
2. 输入标题（可选）和详细内容（可选）
3. 点击 `✓` 确认
4. 确认后显示备注文字和编辑按钮

#### 计划模式备注

1. 在滚轮选择器下方输入备注
2. 点击「+ 工作」或「+ 休息」添加带备注的任务
3. 任务列表中显示 📒 图标表示有备注
4. 点击 📒 图标可查看/编辑/删除备注

#### 统计中显示

备注会在数据统计中按类别分组显示：
- 柱状图按备注类别堆叠显示
- 饼图显示各备注类别的时长占比

## 👥 团队协作指南

### Git 基础操作

```bash
# 查看修改
git status

# 添加修改
git add .

# 提交修改
git commit -m "feat: 添加了新功能"

# 推送到远程
git push
```

### Commit 提交规范

| 前缀 | 用途 | 示例 |
|------|------|------|
| `feat:` | 新功能 | `feat: 添加开机自启动功能` |
| `fix:` | 修复 Bug | `fix: 修复计时器暂停后时间重置的问题` |
| `docs:` | 文档修改 | `docs: 更新 README 安装说明` |
| `refactor:` | 代码重构 | `refactor: 重构计时器模块` |
| `chore:` | 构建/工具相关 | `chore: 更新依赖版本` |
