📖 使用说明

🔄 模式切换

顶部滑块切换 单次模式 和 计划模式

⏱️ 单次模式

• 点击左侧预设设置时间，或用滚轮自定义（1-120分钟）p>

• 点击 "+" 添加新预设，点击 "×" 删除预设

• "工作"/"休息" 按钮切换模式

📝 计划模式

• 滚轮选时间，点击 "+ 工作"/"+ 休息" 添加任务

• 拖拽调整顺序，点击 "×" 删除任务

• 点击"开始"自动依次执行计划

🌱 专注模式与菜园子

• 开启专注模式后参与奖惩机制

• 计时运行时作物会成长

• 切换娱乐应用会收到警告，3次警告作物枯萎

• 点击 🌱 按钮打开菜园子管理作物

🎵 音乐播放器

• 播放控制：播放/暂停、上下首

• 快捷键：Ctrl+↑/↓ 调音量

☁️ 云端登录

• 注册账号或登录已有账号

• Admin 用户可自动获取 DeepSeek API Key

• 普通用户可使用基础功能

🤖 AI规划助手

• 需要 Admin 用户登录

• 点击 🤖 按钮，用自然语言描述需求

• 点击"应用计划"自动添加到计划模式

🤖 AI规划助手

💡 提示：描述您的任务内容、时长需求，AI会为您规划合理的番茄钟计划
